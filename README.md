# Spotify Data Visualization and Analysis

## Project Overview

This project focuses on analyzing and visualizing Spotify music data to uncover patterns, trends, and relationships among various audio features. The analysis was performed using Python, Pandas, Matplotlib, and Seaborn.

The dataset contains over 113,000 Spotify tracks along with information about artists, genres, popularity, danceability, energy, loudness, acousticness, valence, and other musical attributes.

## Dataset Information

* Dataset: Spotify Tracks Dataset
* Records: 113,999
* Features: 20+
* Source: Spotify Music Data

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook
* VS Code

## Data Preprocessing

The following preprocessing steps were performed:

* Loaded the dataset using Pandas
* Removed unnecessary columns
* Checked for missing values
* Handled null values
* Created additional features where required
* Prepared data for visualization

## Key Findings

* Pop-film and K-pop are among the most popular genres.
* Most songs have low to moderate popularity.
* Non-explicit songs outnumber explicit songs.
* Danceability has a weak positive relationship with popularity.
* Most songs are between 2 and 5 minutes long.
* Energy and loudness show a strong positive correlation.
* Energy and acousticness show a strong negative correlation.
* Most songs have moderate valence values, indicating balanced emotional tones.
