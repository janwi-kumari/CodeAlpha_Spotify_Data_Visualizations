# CodeAlpha_Spotify_Data_Visualizations
This project focuses on analyzing and visualizing Spotify music data to uncover patterns, trends, and relationship among various audio features. The analysis was performed using Python, Pandas, Matplotlib and Seaborn.
The dataset contains over 113000 Spotify tracks along with information about artistics, genres, popularity, danceability , energy, loudness, acosticness, valence and other musical attributes.
